# 手動測試清單 — 場地平面圖編輯器 (Konva 基礎)

適用範圍：`/venue` 頁面的 Konva 2D 平面圖編輯器基礎（畫布、格線、比例標示、地板多邊形編輯）。
以下項目為 Playwright 難以判斷的視覺／手感檢查，需人工瀏覽器確認。

## 視覺 / 版面

- [ ] 畫布背景為淺色、乾淨的平面圖風格（非純白、非深色）
- [ ] 每 1 公尺有淺色格線，每 5 公尺有較深的格線，兩者對比清楚可辨識
- [ ] 畫布外圍有明顯邊框，標示 50x50 公尺的可用範圍
- [ ] 左側與上方每 5 公尺有數字刻度標籤（0, 5, 10 … 50），字體在預設視窗大小下清晰可讀
- [ ] 左下角有比例尺（5 公尺線段 + 「5 公尺」文字），位置不與其他元素重疊

## 預設地板多邊形

- [ ] 頁面載入後，畫布中央附近顯示一個 10x10 公尺的正方形地板
- [ ] 地板有半透明填色與清楚的邊框線
- [ ] 4 個頂點皆顯示為可辨識的白底藍框圓形把手

## 拖曳手感

- [ ] 拖曳頂點時，把手跟隨滑鼠移動流暢，無明顯延遲或跳動
- [ ] 放開滑鼠後，頂點吸附至最近的 0.5 公尺格線，肉眼可辨識吸附感
- [ ] 拖曳頂點超出畫布邊界時，頂點停留在邊界上，不會消失或跑到畫布外

## 邊上插入頂點 / 刪除頂點

- [ ] 雙擊任一邊的中點附近，可看到新頂點出現，多邊形形狀合理分裂為兩段
- [ ] 在既有頂點正上方雙擊，不會產生新頂點（無視覺變化）
- [ ] 對頂點按右鍵（多於 3 個頂點時）可刪除該頂點，多邊形重新連接無破圖
- [ ] 只剩 3 個頂點時，嘗試刪除不會有作用，畫面無變化

## 凹多邊形

- [ ] 將某個頂點拖曳至多邊形內部，形成凹多邊形，畫面正常渲染、無錯誤或崩潰

## 視窗大小調整

- [ ] 縮小瀏覽器視窗寬度後，畫布會依比例縮小並維持可互動狀態（拖曳/雙擊/刪除仍正常運作）
- [ ] 將視窗縮到約 360px 寬時，刻度標籤仍可辨識文字內容（若模糊不清，於此記錄）

## Task 2 — 牆壁 / 柱子物件系統

適用範圍：`選取`/`牆壁`/`柱子` 工具列與牆壁、柱子物件的繪製、選取、移動、刪除。
以下為 Playwright 難以判斷的視覺／手感檢查，需人工瀏覽器確認。

### 工具列外觀與狀態

- [ ] 工具列顯示三個模式按鈕（選取／牆壁／柱子）與一個「刪除」按鈕，樣式與現有按鈕一致
- [ ] 頁面載入後「選取」按鈕呈現按下（active/aria-pressed）樣式，其餘兩個未按下
- [ ] 切換模式時，對應按鈕即時呈現按下樣式，其餘按鈕恢復未按下樣式
- [ ] 「刪除」按鈕在未選取任何物件時呈現停用（disabled）樣式；選取物件後立即變為可點擊

### 牆壁繪製手感

- [ ] 進入「牆壁」模式後，滑鼠壓下拖曳時可即時看到半透明的預覽牆壁線段跟隨游標
- [ ] 放開滑鼠後，預覽消失，正式牆壁以深琥珀色實心色塊呈現，厚度明顯可見（非細線）
- [ ] 拖曳距離過短（吸附後起訖點相同）放開滑鼠後，不會產生新牆壁，且仍停留在「牆壁」模式
- [ ] 成功建立牆壁後，自動切回「選取」模式，且新牆壁呈現藍色外框（已選取狀態）

### 柱子繪製手感

- [ ] 進入「柱子」模式後單擊，立即出現一個 0.5x0.5 公尺的中灰色方塊，邊框顏色略深
- [ ] 柱子中心點吸附至 0.5 公尺格線，肉眼可辨識吸附感
- [ ] 放置後自動切回「選取」模式，且新柱子呈現藍色外框（已選取狀態）

### 選取 / 移動 / 刪除手感

- [ ] 「選取」模式下點擊牆壁或柱子，該物件立即顯示藍色外框；點擊空白畫布，外框消失（取消選取）
- [ ] 選取牆壁後，其兩端點顯示與地板頂點相同樣式的白底藍框圓形把手，位置精準對齊牆壁端點
- [ ] 拖曳已選取物件本體（非端點把手）時，整個物件跟隨游標平移，流暢無明顯延遲
- [ ] 拖曳牆壁端點把手時，只有該端點移動，另一端點完全不動
- [ ] 將一端點拖曳到與另一端點重合的位置時，放開滑鼠後端點會彈回原本合法位置（無視覺跳動異常）
- [ ] 拖曳物件靠近畫布邊界時，物件停留在邊界內（含整個物件範圍），不會部分跑出畫布外
- [ ] 按下 Delete 或 Backspace 鍵，已選取物件立即消失；點擊「刪除」按鈕效果相同
- [ ] 未選取任何物件時按 Delete/Backspace 或點擊「刪除」，畫面無任何變化、無錯誤訊息

### 多物件情境

- [ ] 建立多個牆壁與柱子後，個別選取、移動、刪除其中一個，其餘物件外觀與位置不受影響

## Task 3 — 縮放與尺寸標註

適用範圍：柱子縮放把手、柱子/牆壁尺寸標籤（選取/拖曳時顯示）、地板多邊形邊長標籤（常駐顯示）。
以下為 Playwright 難以判斷的視覺／手感檢查，需人工瀏覽器確認。

### 柱子縮放把手外觀與手感

- [ ] 選取柱子後，其矩形四個角落各出現一個白底藍框圓形把手，樣式與地板頂點/牆壁端點把手一致
- [ ] 拖曳任一角落把手時，該角落跟隨滑鼠移動流暢，對角把手完全靜止不動
- [ ] 拖曳中的把手呈現藍色實心樣式，放開後恢復白底
- [ ] 縮放吸附至 0.5 公尺格線，肉眼可辨識吸附感
- [ ] 嘗試將柱子縮小超過下限（0.5 x 0.5 公尺）時，把手停在下限位置，不會繼續縮小或消失
- [ ] 將柱子拖曳/縮放至畫布邊界附近時，縮放會停在邊界內，不會部分跑出畫布外
- [ ] 取消選取柱子後，四個縮放把手立即消失

### 尺寸標籤顯示手感

- [ ] 選取柱子時，柱子附近立即顯示「寬 x 高 公尺」文字標籤（例如「1.5 x 1.0 m」），字體清晰易讀
- [ ] 拖曳柱子縮放把手時，尺寸標籤即時更新，數字與目前柱子大小一致，無延遲感
- [ ] 取消選取柱子後，尺寸標籤立即消失，不殘留
- [ ] 選取牆壁時，牆壁中點附近立即顯示「長度 公尺」文字標籤（例如「3.2 m」）
- [ ] 拖曳牆壁端點把手時，長度標籤即時更新，無延遲感
- [ ] 取消選取牆壁後，長度標籤立即消失，不殘留
- [ ] 柱子與牆壁的標籤位置不與物件本身或把手重疊，清晰可讀

### 地板邊長標籤（常駐顯示）

- [ ] 頁面載入後，地板多邊形每一條邊附近都顯示一個「長度 公尺」文字標籤，無需選取
- [ ] 拖曳地板頂點時，相鄰兩條邊的長度標籤即時跟著更新，無延遲感
- [ ] 在邊上雙擊插入新頂點後，邊長標籤數量正確增加為對應的邊數
- [ ] 右鍵刪除頂點後，邊長標籤數量正確減少
- [ ] 所有標籤數字皆為小數點後一位，並帶有「m」單位後綴，格式一致
